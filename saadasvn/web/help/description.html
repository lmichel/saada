Database description
This page displays the content of the file SaadaDB_Home/web/help/description.html.
Put the HTML description of your archive in this file and deploy the Web application again.
Don't write anything directly into the the Tomcat WebApp directory because this directory is overridden each time the SaadaSDB Web application is deployed
Be careful to target links to a new browser tab (<a href=... target=blank\>) in order to not overridden the current SaadaDB session.
